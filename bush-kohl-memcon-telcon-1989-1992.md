# Bush–Kohl Memcons and Telcons (1989–1992)

A structured research page for declassified memoranda and telephone conversations between President George H. W. Bush and Chancellor Helmut Kohl, intended as a companion aid for FRUS-era documentary tracking.

This page follows a repository-style documentation format with explicit source anchors, metadata fields, and a reproducible update workflow.

## Research Scope

The current list focuses on currently discoverable, publicly posted Bush–Kohl records (1989–1992) with explicit citation notes pointing to Bush Presidential Library holdings (including Brent Scowcroft Collection references).

Planned expansion:
- NARA Catalog descendants under NAID **4522156** (file-unit level where available)
- Bush Library FOIA release inventories (including FOIA 2009-0275-S)
- Additional Bush Library Memcons/Telcons finding aids that may include overlooked Bush–Kohl traffic

## Record Metadata Model

Each record should be normalized to the following fields:

- `date` (ISO date for sorting)
- `type` (memcon/telcon)
- `title` (source document title)
- `direction` (Bush → Kohl, Kohl → Bush, or in-person meeting)
- `participants`
- `holding` (collection/subseries language from source citation)
- `release_status` (e.g., MDR, FOIA)
- `source_page`
- `official_pdf` (if available)
- `naid` (if available)
- `notes`

## Update Workflow (Recommended)

1. Add newly confirmed records to the chronological table.
2. Verify date normalization (`YYYY-MM-DD`) and direction labels.
3. Preserve exact collection language from each citation note.
4. Attach NAID and official PDF links when discoverable.
5. Keep entries sorted by date ascending.

## Quality Control Checklist

- Confirm no duplicate entries by date/title pair.
- Separate phone calls from in-person memcons in `type`.
- Flag uncertain attribution in `notes` before promotion to main list.
- Track unresolved archival leads in a dated research log section (if added later).
